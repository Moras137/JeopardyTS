Schnellstart

  1. Oeffne das Dashboard unter /create.html.
  2. Klicke + Neues Quiz erstellen und trage Titel/Kategorien/Fragen ein.
  3. Speichere das Quiz.
  4. Klicke auf Starten beim gewünschten Quiz.
  5. Im Hostfenster wird ein Raumcode angezeigt. Das Board öffnest du über den Board-Link.
  6. Spieler treten über /player.html?room=CODE bei (oder per QR).
Hinweis: Die App arbeitet mit Reihum-Turns. Nach Bewertung ist der nächste Spieler dran.

Dashboard

  • Quiz suchen: Filtert die Liste nach Titel.
  • Starten: Öffnet Hostansicht für ein Quiz.
  • Bearbeiten: Öffnet den Editor für das Quiz.
  • Löschen: Entfernt das Quiz aus der Liste.
  • Anleitung: Diese Seite ist direkt vom Dashboard aus erreichbar.

Quiz-Editor

Grundaufbau

  • Quiz-Titel vergeben
  • Kategorien anlegen
  • Pro Kategorie Fragen mit Punktwert erstellen

Medien

  • Frage und Antwort können eigene Medien haben (Bild/Audio/Video).
  • Medien kannst du per Drag-and-drop direkt in die Upload-Bereiche ziehen.
  • Map-Fragen unterstützen Zielpunkt, Radius oder Zone.

Speichern und Laden

  • Nach Änderungen speichern.
  • Bereits angelegte Quizze können jederzeit erneut bearbeitet werden.

Board-Vorschau

  • Im Board-View siehst du die Struktur wie später auf dem Spielbrett.
  • Du kannst Fragen auch direkt in der Boardansicht anklicken und bearbeiten.
  • In der Boardansicht kannst du Medien ebenfalls per Drag-and-drop anfügen.
  • In der Kachelansicht ziehst du die Datei direkt auf die gewünschte Frage-Kachel: linke Hälfte = Frage-Medien, rechte Hälfte = Antwort-Medien.
  • Auch Medien lassen sich dort nachträglich ändern, austauschen oder entfernen.
  • Hilft beim schnellen Check von Punkten und Kategorieaufbau.

Fragetypen

Standard Klassisch

  • Frage + Lösung, optional Medien.
  • Host bewertet Antwort, danach nächster Spieler.

List Hinweis-Modus

  • Hinweise können schrittweise aufgedeckt werden.
  • Gut für Begriffsrätsel mit steigender Leichtigkeit.

Elemination Mdl. Antworten + Aufdecken

  • Spieler antworten mündlich, Host bewertet richtig/falsch.
  • Host deckt passende Antworten einzeln auf.
  • Falsche Antwort kann zur Elimination führen.

Map

  • Spieler setzen Marker auf Karte.
  • Host löst gesammelt auf, Punkte nach Regel (Radius/Nähe/Zone).

Estimate

  • Alle geben eine Zahl ab.
  • Auswertung nach Nähe zum Zielwert.

Pixel

  • Bild wird stufenweise deutlicher.
  • Host kann Verlauf steuern und bewertet Antwort.

Freetext

  • Freie Texteingabe durch Spieler.
  • Host vergibt Korrekt/Inkorekt und Punkte.

Hosting Ablauf

  1. Quiz im Dashboard starten.
  2. Hostansicht zeigt Raumcode und Board-Link.
  3. Board auf separatem Bildschirm/Fenster öffnen.
  4. Spieler treten bei.
  5. Host wählt Frage und moderiert Bewertung/Steuerung.
  6. Nach jeder Bewertung geht der Turn weiter.
  7. Am Ende kann die Siegerehrung gestartet werden.

Reihum und Turnlogik

  • Es wird nicht mehr gebuzzert, sondern in Reihenfolge gespielt.
  • Der aktive Spieler wird auf Host, Board und in Spieleransichten angezeigt.
  • Der Host kann den aktuellen Spieler in der Host-Spielerliste per Doppelklick setzen.
  • Nach Bewertung (richtig oder falsch) wechselt der Turn zum nächsten Spieler.

Elemination-Modus im Detail

  1. Frage starten, erster Spieler ist dran.
  2. Host bewertet Antwort mit Richtig oder Falsch.
  3. Passende Antwort wird über die Aufdecken-Buttons einzeln gezeigt.
  4. Bei Falsch kann der Spieler eliminiert werden.
  5. Turn geht an den nächsten nicht eliminierten Spieler.
  6. Am Rundenende werden je nach Zustand Punkte vergeben und restliche Antworten aufgedeckt.
Wichtig: Aufgedeckte Antworten erscheinen direkt auf dem Board.

Board- und Spieleransicht

Board

  • Zeigt Fragen, Medien, Aufdeckungen und Punktestand.
  • Markiert den aktuell aktiven Spieler.

Spieleransicht

  • Zeigt für alle sichtbar, wer gerade dran ist.
  • Eigener Turn wird deutlich hervorgehoben.
  • Map/Estimate/Freetext bekommen eigene Eingabebereiche.

Tipps für gute Quizze

  • Starte leicht und steigere Schwierigkeit/Punkte pro Kategorie.
  • Mische Fragetypen für Tempo und Abwechslung.
  • Verwende kurze, klare Frageformulierungen.
  • Teste Medien vor dem Live-Spiel (Dateien und Lautstärke).
  • Bei Elemination: Antwortliste klar und eindeutig halten.

FAQ

Warum sieht ein Spieler keinen Turnwechsel?

Prüfe die Verbindung und lade die Seite neu. Danach sollte die Anzeige wieder aktuell sein.

Wie öffne ich das Board schnell?

In der Hostansicht auf den Board-Link klicken oder URL mit Raumcode nutzen.

Kann der Host den aktiven Spieler korrigieren?

Ja. In der Host-Spielerliste den gewünschten Spieler doppelklicken.
